# Rebel Leaders — RPG Redesign: "The Quest-Giver's Study"

## CHOSEN DIRECTION

### Design Movement
**16-bit RPG Nostalgia meets Philosophical Depth** — SNES-era RPG visual language applied to thought-leadership. The dissonance IS the brand.

### Core Principles
1. The Medium IS the Message — pixel art framing says "you've been playing the wrong game"
2. Disarming Before Delivering — nobody expects divided brain theory in a SNES dialogue box
3. Warm, Not Dark — cozy RPG inn, not a cathedral
4. Scroll = Journey — the page unfolds like an RPG storyline

### Color Palette (from the actual office)
- Deep Forest Green (#1a3a2a) — wall anchor
- Warm Wood Brown (#6b4c2a) — shelves, warmth
- Tolkien Gold (#d4a843) — accents, highlights
- Leather Red (#8b3a3a) — emphasis
- Parchment Cream (#f0e6d0) — text
- Pixel Black (#0d1a12) — deepest background

### Typography
- Press Start 2P — pixel headings, nav, UI labels
- Cormorant Garamond — body text (timeless ideas beneath pixel wrapper)
- Outfit — buttons, forms, secondary UI

### Signature Elements
1. Pixel art office scene as hero
2. RPG dialogue boxes for key quotes
3. Typewriter text animation
4. Pixel art section dividers and icons
